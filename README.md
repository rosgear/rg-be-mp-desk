## <img src="https://raw.githubusercontent.com/rosgear/rg-be-mp-desk/refs/heads/main/assets/images/icon.svg" width="64px" height="64px" align="absmiddle"> Расширение модуля «Панель приложений Маркетплейс»

Панель Маркетплейса приложений предоставляет возможность выбора и установки готовых программных решений для веб‑приложения RosGear, включая модули, расширения, виджеты, плагины и темы оформления.

Расширение модуля входит в состав Панели управления GePanel и cлужит расширением для [модуля «Маркетплейс приложений»](https://github.com/rosgear/rg-be-mp).

### Установка

Для добавления расширения модуля в ваш проект, вы можете просто выполнить команду ниже:

```
$ composer require rosgear/rg-be-mp-desk
```

или добавить в файл composer.json вашего проекта:
```
"require": {
    "rosgear/rg-be-mp-desk": "*"
}
```
или скачать архив на [странице расширения модуля](https://rosgear.ru/component/rg-be-mp-desk/) в каталоге приложений RosGear.

После добавления расширения модуля в проект выполните его установку в редакцию веб‑приложения с помощью Панели управления GePanel.

## <img src="https://raw.githubusercontent.com/rosgear/rg-be-mp-desk/refs/heads/main/assets/images/icon.svg" width="64px" height="64px" align="absmiddle"> Module extension «Marketplace Application Panel»

The Application Marketplace panel provides the ability to select and install ready-made software solutions for the RosGear web application, including modules, extensions, widgets, plugins, and themes.

The module extension is part of the GePanel Control Panel and serves as an extension for the [«Application Marketplace» module](https://github.com/rosgear/rg-be-mp).

### Installation

To add the module extension to your project, you can simply run the command below:

```
$ composer require rosgear/rg-be-mp-desk
```

or add to your project's composer.json file:
```
"require": {
    "rosgear/rg-be-mp-desk": "*"
}
```
or download the archive from the [module extension page](https://rosgear.ru/component/rg-be-mp-desk/) in the RosGear application catalog.

After adding the module extension to the project, install it into the web application edition using the GePanel Control Panel.
